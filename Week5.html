Week 5: Reflectie & Beslissingen

Doelen

Het doel van deze week was om mijn ontwerp uit Figma om te zetten naar HTML en CSS. Ik wilde de basisstructuur van de website opbouwen, werken met containers, en mijn gekozen kleuren en lettertypes uit Figma toepassen in de code. Daarnaast wilde ik leren hoe ik mijn project online kan zetten via GitHub en Netlify.

Wat heb ik gedaan?

  • De HTML-structuur opgezet met verschillende pagina’s per week.
  • Een navigatiebalk toegevoegd zodat alle weken met elkaar verbonden zijn.
  • Lettertypes uit Figma geïmporteerd via Google Fonts.
  • Een achtergrondkleur en basisopmaak toegevoegd.
  • Mijn project gepubliceerd via GitHub en Netlify.

Redenen voor mijn keuzes

Ik koos voor een rustig en overzichtelijk ontwerp, met voldoende witruimte. Dat sluit aan bij de visuele stijl van mijn Figma-ontwerp. De gele kleur heb ik meegenomen in mijn HTML/CSS om consistentie te bewaren.

Lessen & Next steps

Deze week leerde ik dat een duidelijke HTML-structuur essentieel is. Ik begrijp nu beter hoe ik lettertypes en kleuren één-op-één kan vertalen uit Figma. Volgende week wil ik experimenteren met lay-out, grid en spacing zodat de website nog professioneler oogt.
